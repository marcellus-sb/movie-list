# movie-list

This project is a playground to exercise and test Swift codes. The idea is develop new features over time, generate discussions and knowledge growth.

## Features

- [x] List Movies
- [x] Movie Detail
- [x] Favorites
- [x] Order List
- [ ] Change Theme
- [ ] Reminder
- [ ] Share
- [ ] Youtube Trailer
- [ ] Chromecast
